# Release v1.1.2

## Summary

Security Gate workflow improvements, README updates, and CI fixes.

## Highlights

- Security Gate workflow: Bandit skip list expanded, Gitleaks configuration fixed, safety check command updated
- README: Connected repositories section added documenting plugins
- CI fixes: Shell injection prevention, markdown quality improvements

## Changes

### Security

- Bandit: tests directory excluded from scan, skip list expanded (B101, B105, B106, B107, B108, B310, B404, B603, B607)
- Gitleaks: Added fetch-depth: 0 for full git history
- Semgrep: Shell injection prevention using environment variables
- Safety: Fixed command syntax (--full-report)
- Removed security-posture job per user request
- Inlined reusable workflow jobs

### CI/CD

- YAML flow mapping to block mapping conversion
- Fixed e2e job conditional logic
- Fixed printf format string issue
- Added PYTHONPATH=. for memx-resolver tests
- Fixed permissions/concurrency configuration structure

### README

- Added Connected Repositories section documenting agent-taskstate and memx-resolver plugins
- Organized multilingual READMEs

### Tests & Quality

- Added 12 test files improving coverage to 87.75%
- Fixed Markdownlint errors (MD022, MD024, MD031, MD032, MD040, MD047)
- Fixed CodeQL code scanning alerts
