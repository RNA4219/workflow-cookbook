# データセット登録台帳

セキュリティレビューおよび再現性確保のため、取得した学習・検証用データセットのメタデータを以下のテンプレートで管理する。ハッシュ値は展開後のアーカイブに対して `sha256sum <file>` で算出し、変更があった場合は新行を追加して履歴を残す。

## 記録テンプレート

| データセット名 | バージョン / タグ | 取得元 (URL / リポジトリ) | ハッシュ値 (SHA256) |
| --- | --- | --- | --- |
|  |  |  |  |
